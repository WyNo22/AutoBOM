# RETEX — retours d'expérience

Journal des écueils, mauvaises décisions, leçons apprises, à enrichir au fil de l'eau.

Format : un fichier par retex, nommé `YYYY-MM-DD-slug.md`. Garder le contenu **court** : Symptôme → Cause → Correction → Leçon.

## Index

(vide — Sprint 0 vient de démarrer)

## Template

```md
# 2026-01-15 — Title

**Symptôme** : ce qui a planté ou pris trop de temps.
**Cause racine** : la vraie raison sous-jacente.
**Correction** : ce qu'on a fait pour fixer.
**Leçon** : règle générale à appliquer la prochaine fois.
```
